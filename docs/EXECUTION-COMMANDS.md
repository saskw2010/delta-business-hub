# Execution Commands

The execution board is maintained in [`../EXECUTION-BOARD.md`](../EXECUTION-BOARD.md).

Use these compact commands in chat:

- `أضف فكرة: <text>` — capture a new user idea.
- `نفذ <ID>` — approve and execute a pending item immediately.
- `اعتمد <ID>` — approve a recommendation or suggestion and move it to execution.
- `أوقف <ID>` — put an item on hold.
- `ألغِ <ID>` — cancel an item.
- `اعرض اللوحة` — show recommendations, ideas, pending work, suggestions, blockers, and completed items.

The completion response is limited to the actual result, verification, link, and genuine blockers.
